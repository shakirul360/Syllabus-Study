CSS Selectors - Teaches us how to select items and add style rules for them.

Display and positioning - Teaches us how to position items relatively, absolutely and fixed wise. Z-index lesson included too.

The Box Model - lesson on how The Box Model works and how we can position items inside the container box.

Flexbox - Lesson on how to create a block level flex container which takes the full width of its parent container and customise it accordingly.

Grid - How to create a grid, and customise it using rows and columns.

Transition - customising the property of elements and setting delays.
